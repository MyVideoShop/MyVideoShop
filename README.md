# Automatischer Bot mit Session-Management

Dieser Bot loggt sich täglich auf einer Website ein und erledigt automatisch Aufgaben.

## Features
- Automatischer täglicher Login
- Session-Cookie-Speicherung (keine erneute Eingabe nötig)
- Web-Interface für manuellen Login bei Session-Ablauf
- Verschlüsselte Speicherung der Cookies
- Läuft auf Render als Web-Service
